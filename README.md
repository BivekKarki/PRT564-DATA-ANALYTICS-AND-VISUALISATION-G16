PRT564 Data Analytics & Visualisation

Assessment 2 — Regression Pipeline

Project Overview
This project develops a complete data analytics and machine learning pipeline to analyse and predict the energy efficiency (AEER) of household air conditioners.

The workflow includes:
Data loading and cleaning
Integration of external climate data
Feature engineering and preprocessing
Exploratory Data Analysis (EDA)
Regression modelling (OLS & Random Forest)
Model evaluation and statistical testing
